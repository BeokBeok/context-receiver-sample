# context-receiver-sample

## 환경 설정
1. 코틀린 버전 설정 
    - 1.6.20-M1 이상
2. freeCompilerArgs 설정
    - "-Xcontext-receivers" 추가
3. 코틀린 플러그인 버전 설정
    - Android Studio -> Preferences -> Languages & Frameworks -> Kotlin
    - `Update channel`에 `Early Access Preview` 설정
